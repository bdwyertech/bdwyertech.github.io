Cisco IOS - show run linenum

November 11, 2011 One-minute read
Networking

So I discovered a nifty little trick recently for Cisco IOS.  You can get line numbers to appear alongside your configuration when doing a show run!  I found this to be awesome, although I do not think it is available on all IOS since my 3550’s running 12.2(44) do not have this feature.  My 1841 running the latest 15.1(4) is the device I used to discover it.  Rather than doing a simple show run, give show run linenum a shot.  It is pretty nifty, check it out below.
