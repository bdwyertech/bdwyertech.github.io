Passed 70-647 MCITP Server 2008 Enterprise Administrator

I passed the MCITP Server 2008 Enterprise Administrator exam earlier today.  As the second generation of Microsoft certification, I am hoping that it correlates well with the retired and well-respected MCSE.  It was an interesting process and I feel I learned a good bit about Microsoft enterprise solutions, especially the new technologies that earlier iterations of their products did not have.

I used Microsoft training materials for the all exams as well as the Sybex Windows Server 2008 R2 Complete.  In my home virtual lab, I also completed the practice tasks and constructed scenarios which I was not familiar with.  Now, time to take some time to clear my head and get focused on Cisco again!
